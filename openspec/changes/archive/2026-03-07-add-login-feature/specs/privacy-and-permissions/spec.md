## ADDED Requirements

### Requirement: 인증 전면 게이트 적용
시스템은 인증이 필요한 기능에 대해 비인증 사용자의 접근을 차단해야 한다. (MUST)

#### Scenario: 비로그인 사용자 접근
- **WHEN** 비로그인 사용자가 보호 API 또는 보호 화면에 접근을 시도한다
- **THEN** 시스템은 접근을 거부하고 `AUTH_REQUIRED` 오류 코드를 반환해야 한다

### Requirement: 인증 후 소유권 검증 유지
시스템은 인증 성공 이후에도 사용자 소유권 검증 규칙을 유지해야 한다. (MUST)

#### Scenario: 인증되었지만 타인 데이터 접근 시도
- **WHEN** 로그인된 사용자가 본인 소유가 아닌 민감 데이터에 접근을 시도한다
- **THEN** 시스템은 요청을 거부하고 소유권 오류 코드를 반환해야 한다

### Requirement: 인증 실패 감사 로그
시스템은 인증 실패와 권한 거부 이벤트를 요청 추적 정보와 함께 감사 로그에 기록해야 한다. (SHALL)

#### Scenario: 인증 실패 이벤트 기록
- **WHEN** 로그인 실패 또는 세션 만료로 요청이 거부된다
- **THEN** 시스템은 사용자 식별자(가능한 경우), 요청 ID, 실패 사유를 감사 로그에 기록해야 한다
