## ADDED Requirements

### Requirement: 메뉴 기반 기능 페이지 분리
시스템은 공통 앱 셸 메뉴(홈/기념일/일정/설정)에 대응하는 독립 페이지를 제공해야 한다. (MUST)

#### Scenario: 메뉴 선택으로 기능 페이지 이동
- **WHEN** 사용자가 사이드바 또는 모바일 드로어에서 메뉴 항목을 선택한다
- **THEN** 시스템은 선택한 메뉴의 대응 페이지로 이동해야 한다

### Requirement: 페이지별 책임 분리
시스템은 기능을 페이지 책임에 맞춰 분리해야 하며, 한 페이지가 모든 기능을 독점하지 않아야 한다. (MUST)

#### Scenario: 기념일 기능 접근
- **WHEN** 사용자가 기념일 메뉴로 이동한다
- **THEN** 시스템은 기념일 조회/추가/수정/삭제 흐름을 해당 페이지에서 제공해야 한다

#### Scenario: 일정 기능 접근
- **WHEN** 사용자가 일정 메뉴로 이동한다
- **THEN** 시스템은 월간 타임라인과 동기화 안내를 해당 페이지에서 제공해야 한다

#### Scenario: 설정 기능 접근
- **WHEN** 사용자가 설정 메뉴로 이동한다
- **THEN** 시스템은 권한 요청 및 계정 관련 조작(예: 로그아웃)을 해당 페이지에서 제공해야 한다
