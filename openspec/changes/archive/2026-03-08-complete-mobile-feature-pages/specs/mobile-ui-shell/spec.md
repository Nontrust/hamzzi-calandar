## MODIFIED Requirements

### Requirement: 페이지 분리 이후 공통 UI 셸 일관성 유지
시스템은 페이지가 분리되어도 공통 셸(사이드바/헤더/모바일 드로어)의 정보 구조와 시각 규칙을 일관되게 유지해야 한다. (MUST)

#### Scenario: 경로 변경 시 헤더 문맥 반영
- **WHEN** 사용자가 홈 외 다른 메뉴 페이지로 이동한다
- **THEN** 시스템은 현재 페이지 문맥에 맞는 헤더 타이틀/보조 문구를 표시해야 한다

#### Scenario: 모바일 드로어 메뉴 사용
- **WHEN** 모바일 사용자가 삼선 버튼으로 드로어를 열고 메뉴를 선택한다
- **THEN** 시스템은 동일한 셸 규칙을 유지한 채 선택한 페이지로 이동해야 한다
