# OpenSpec 문서 구조

이 저장소는 OpenSpec `spec-driven` 스키마 기준으로 운영한다.

## 표준 경로
- `openspec/specs/<capability>/spec.md`: 메인 capability 스펙
- `openspec/changes/<change-name>/`: 변경안 산출물(proposal/design/tasks + delta specs)
- `openspec/changes/archive/`: 완료 변경안 아카이브
- `openspec/config.yaml`: OpenSpec 메인 설정

## 현재 메인 스펙(capabilities)
- `user-role-selection`
- `study-execution`
- `memory-reward-system`
- `exam-schedule-integration`
- `ai-interview-coaching`
- `music-playlist-links`
- `privacy-and-permissions`

## 레거시 문서 보관
기존 초안 문서는 아래 경로에 보존한다.
- `openspec/docs/legacy/spec/`
- `openspec/docs/legacy/tasks/`
