# MVP 개발 계획

## 마일스톤
M1. 스터디 세션 + TODO + 이월 + 달성률 계산  
M2. 추억 보상(잠금/해금) + 해금 규칙  
M3. 시험 일정 API 수집 + 정규화 + Google Calendar 이벤트 생성  
M4. AI 모의면접(채팅) + 리포트 저장  
M5. 유튜브 뮤직 탭(링크 저장/실행)  

## 테스트 체크리스트
- 세션 시작/종료 예외(앱 강제 종료/네트워크 없음)
- TODO 이월: 내일/특정 날짜로 이동
- 달성률 계산 정확성(0/100/중간값)
- 보상 해금 조건 정확성 및 하루 제한
- 시험 일정 중복 제거(기관/제목/기간 동일)
- 캘린더 동기화: 생성 vs 업데이트 처리
- 모의면접 리포트 필드 누락 없이 생성
