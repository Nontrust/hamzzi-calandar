# 제품 비전

## 제품 개요
커플(나 + 여자친구)용 **공기업 합격 지원 앱**.  
학습 플래너(과목별 시간표 자동 생성)는 제외하고, 다음에 집중한다.

- **스터디 세션 기록**(시작/종료, TODO, 달성 체크, 미달성 이월)
- **목표 달성 기반 보상**(추억 사진/문구 잠금 해제)
- **공기업 채용/시험 일정 자동 등록**(Open API 수집 → Google Calendar 동기화)
- **AI 모의면접**(대화형 + 리포트)

## 핵심 사용 흐름(Core Loop)
1. 오늘 **스터디 시작**(오늘의 TODO 목표 입력)
2. **스터디 종료**(TODO 달성/미달성 체크)
3. **달성률 계산** + 미달성 TODO는 원하는 날짜로 **이월**
4. 달성률 조건 충족 시 **추억(사진/문구) 잠금 해제**
5. 시험/원서접수/발표 일정은 자동으로 캘린더에 등록
6. 면접 준비는 AI 모의면접으로 반복

## 주요 사용자
- **사용자 A(나)**: 설정/관리/응원 문구 작성, 추억 업로드, 기관/키워드 설정
- **사용자 B(여자친구)**: 스터디 수행, TODO 체크, 모의면접 진행

## 성공 지표(예시)
- 일/주 스터디 세션 수
- 일/주 평균 달성률
- 미달성 TODO가 7일 내 완료되는 비율
- 원서접수 마감/시험/발표 일정 누락 0건
