# 범위(Scope)

## 포함(In Scope) - MVP
- 스터디 세션: 시작/종료, TODO 작성/체크, 미달성 이월
- 달성률: 세션/일/주 단위 퍼센트 계산 및 표시
- 보상(추억 보관함): 달성 조건 충족 시 사진/문구 열람(잠금 해제)
- 시험 일정: 채용/시험 관련 일정 자동 수집(Open API 최소 1개) + Google Calendar 이벤트 생성
- AI 모의면접: 대화형 면접 + 종료 리포트 저장
- 유튜브 뮤직: 플레이리스트 링크 저장 + 실행 탭 제공(인앱 브라우저)

## 제외(Out of Scope) - 현재
- 학습 플래너(과목별/시간표 자동 생성)
- 기출/강의/자료 저장소
- 커뮤니티/스터디 모집/공유 게시판
- 완전한 음악 재생 제어(공식 YouTube Music API 부재로 링크 기반 실행 중심)
