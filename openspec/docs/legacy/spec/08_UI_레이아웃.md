# UI 레이아웃 & 반응형

## 타겟
- iOS 모바일(세로) 우선
- 15.6인치 노트북(와이드) 대응

## 내비게이션 구조
### 모바일(하단 탭)
- 홈
- 세션
- 캘린더
- 면접(AI)
- 추억(보상)
- 뮤직

### 데스크톱(좌측 사이드바)
- 위와 동일 메뉴 + 상단 빠른 액션
  - `세션 시작`
  - `TODO 추가`
  - `캘린더 동기화`

## 주요 화면
- 홈: 오늘 달성률, 세션 시작 버튼, 보상 진행률
- 세션: TODO 리스트 + 종료 플로우(달성 체크/이월)
- 캘린더: 시험 일정 + TODO 이월 관리
- 면접: 모드/페르소나 선택 → 채팅 → 리포트
- 추억: 잠금/해금 그리드, 해금 조건 표시
- 뮤직: 플레이리스트 링크 리스트
