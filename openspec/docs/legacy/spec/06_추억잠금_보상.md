# 추억(사진/문구) 잠금 해제(보상) 설계

## UX 원칙
- 보상 보관함은 **잠금(🔒)** 상태가 기본
- 조건 달성 시 애니메이션/효과로 해금(✅)
- 잠금 상태에는 “현재 달성률 X% / 목표 Y%”를 명확히 표시

## 해금 규칙(MVP)
- **일일 달성률 >= 임계치** → 보상 1개 해금
- **주간 평균 달성률 >= 임계치** → 스페셜 보상 해금(선택)

## 부정 방지/품질 기준
- 세션 시간이 너무 짧으면(예: 20분 미만) 달성률 집계에서 제외(옵션)
- 하루 해금 개수 제한(예: 1~2개)
- 동일 보상 중복 해금 방지

## 보상 콘텐츠
- 사진 1장 + 문구 1줄(기본)
- 문구만/사진만도 가능
- 업로드/수정 권한: 기본은 사용자 A(나)
